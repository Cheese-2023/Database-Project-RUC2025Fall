# 贫困县数据分析说明

## 脚本功能

`analyze_poverty_counties.py` 脚本用于从中国县域数据库6.0的ARIMA填补表中筛选出832个国家级贫困县，并进行统计分析。

## 主要功能

1. **数据加载**
   - 从 `中国县域数据库6.0版.xlsx` 的 "ARIMA填补(慎用)" sheet 中加载县域数据
   - 从 `832个国家级贫困县摘帽情况分省分年统计.xlsx` 中加载贫困县摘帽信息

2. **县名匹配**
   - 支持直接匹配、模糊匹配和部分匹配三种方式
   - 成功匹配了827个贫困县（共832个，匹配率99.4%）

3. **统计分析**
   - 基本信息统计（贫困县总数、数据年份范围等）
   - 按年份统计
   - 按省份统计
   - 按摘帽年份统计
   - 经济指标分析（GDP、财政收入、居民收入等）
   - 人口指标分析
   - **摘帽前后对比分析**（摘帽前后3年的经济指标变化）

4. **结果输出**
   - 筛选后的贫困县数据（Excel格式）
   - 贫困县代码列表（文本格式）
   - 分析报告（文本和Excel格式）

## 使用方法

```bash
cd data-scripts
python3 analyze_poverty_counties.py
```

## 输出文件

所有结果保存在 `data/poverty_analysis_output/` 目录下：

1. `poverty_counties_data_YYYYMMDD_HHMMSS.xlsx` - 筛选后的贫困县完整数据
2. `poverty_county_codes_YYYYMMDD_HHMMSS.txt` - 贫困县代码列表
3. `poverty_analysis_report_YYYYMMDD_HHMMSS.txt` - 文本格式分析报告
4. `poverty_analysis_report_YYYYMMDD_HHMMSS.xlsx` - Excel格式分析报告（包含多个sheet）

### 基本信息
- **贫困县总数**: 827个（成功匹配）
- **数据年份范围**: 2000-2023年
- **数据记录总数**: 约19,848条

- 四川省：66个
- 贵州省：65个
- 甘肃省：57个
- 陕西省：56个

- 2016年：3个
- 2017年：11个
- 2018年：282个
- 2019年：339个
- 2020年：52个

### 摘帽前后对比分析（关键发现）

通过对摘帽前后3年的数据对比，发现贫困县摘帽后各项经济指标显著提升：

1. **地区生产总值**
   - 平均增长率：37.7%
   - 摘帽前平均：68.96亿元
   - 摘帽后平均：87.87亿元

2. **人均地区生产总值**
   - 平均增长率：58.6%
   - 摘帽前平均：27,300元/人
   - 摘帽后平均：41,174元/人

3. **地方财政一般预算收入**
   - 平均增长率：44.4%
   - 摘帽前平均：3.76亿元
   - 摘帽后平均：5.25亿元

4. **城镇居民人均可支配收入**
   - 平均增长率：33.5%
   - 摘帽前平均：22,550元
   - 摘帽后平均：30,012元

5. **农村居民人均可支配收入**
   - 平均增长率：54.0%
   - 摘帽前平均：7,367元
   - 摘帽后平均：11,246元

## 注意事项

1. 有5个贫困县未能成功匹配，可能是由于：
   - 县名在数据源中不一致
   - 县名变更（如撤县设市）
   - 数据缺失

2. 摘帽前后对比分析仅针对有完整摘帽年份信息的县（823个）

3. 分析结果基于ARIMA填补的数据，可能存在一定的数据质量差异
