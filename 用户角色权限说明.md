# 用户角色权限说明

## 角色定义

### 1. ADMIN（系统管理员）
- **权限范围**：全部权限
- **功能**：
  - 风险监控大屏
  - AI助手
  - 风险分析
  - 预警管理
  - 数据管理
  - SQL操作
  - 成果展示
  - 风险参数调整

### 2. USER（普通用户）
- **权限范围**：基础查看权限
- **功能**：
  - 风险监控大屏

### 3. VIP（VIP用户）
- **权限范围**：USER权限 + AI助手
- **功能**：
  - 风险监控大屏
  - AI助手

### 4. RISK_ANALYST（风险分析员）
- **权限范围**：VIP权限 + 数据增删改 + 风险参数调整 + 全部页面
- **功能**：
  - 风险监控大屏
  - AI助手
  - 风险分析
  - 预警管理
  - 数据管理（可调整风险参数）
  - SQL操作
  - 成果展示

### 5. DATA_MAINTAINER（数据维护员）
- **权限范围**：VIP权限 + 数据增删改 + 全部页面
- **功能**：
  - 风险监控大屏
  - AI助手
  - 风险分析
  - 预警管理
  - 数据管理（不可调整风险参数）
  - SQL操作
  - 成果展示

## 示例用户

系统已预置以下示例用户：

- `admin` / `admin123` - 系统管理员
- `user1` / `user123` - 普通用户
- `vip1` / `vip123` - VIP用户
- `analyst1` / `analyst123` - 风险分析员
- `maintainer1` / `maintainer123` - 数据维护员

## 注意事项

1. **SQL操作安全**：系统已内置SQL安全检查，禁止执行DROP、TRUNCATE、ALTER等危险操作
2. **风险参数调整**：只有风险分析员和管理员可以调整风险计算参数
3. **数据修改权限**：风险分析员和数据维护员都可以进行数据增删改，但只有风险分析员可以调整风险参数
